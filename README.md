# User Management System

This project is a dynamic CRUD application built using:

- React (Ant Design UI)
- PHP (CodeIgniter Backend)
- MySQL Database

---

## Features

✔ Add User  
✔ Edit User  
✔ Delete User  
✔ Dynamic Fields Support  
✔ Search & Filter  
✔ JSON-based Extra Fields
